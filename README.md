# 🌐 Aetheris Core

> **Framework de coordenação descentralizada para a gestão de ativos e infraestrutura sintética.**

## 📖 Visão Geral

O **Aetheris Core** é uma organização autônoma projetada para governar o ecossistema Aetheris. Através de contratos inteligentes imutáveis, permitimos que detentores de tokens gerenciem uma tesouraria coletiva e direcionem o desenvolvimento de protocolos de próxima geração sem intermediários.

## 🛠 Tech Stack

- **Blockchain:** Arbitrum One / Base (L2 para baixo custo)
- **Custódia:** [Aetheris Vault](https://safe.global/) (Gnosis Safe Custom)
- **Governança:** [Snapshot](https://snapshot.org/) & [Tally](https://www.tally.xyz/)
- **Camada Social:** Discord (Operacional) e Fórum Aetheris (Estratégico)

## 🏛 Estrutura de Governança

O Aetheris utiliza o modelo de **Propostas de Evolução Aetheris (AEPs)**.

1. **Sinalização:** Discussão inicial no canal `#aether-labs` do Discord.
2. **AEP Formal:** Documentação detalhada no Fórum (mínimo 4 dias de debate).
3. **Snapshot (Soft Vote):** Validação da temperatura da comunidade.
4. **On-chain Execution:** Votação final que aciona o _Timelock_ para mudanças no código.

## 🚀 Como Participar

1. **Token de Governança:** Obtenha `$AETHR` via instâncias oficiais.
2. **Portal de Governança:** Conecte sua wallet em `governance.aetheris.io`.
3. **Contribuição:** Acesse o quadro de tarefas (Bounty Board) para ganhar recompensas em $AETHR.
